# Cue AI - Sales Intelligence Platform

A modern landing page for Cue AI, a real-time intelligence platform for sales conversations.

## Features

- **Hero Section**: Main value proposition with CTAs and trust badges
- **Why Cue AI**: Overview of the platform benefits
- **Before Call Section**: Pre-call research & context features
- **During Call Section**: Live meeting intelligence with visual chart
- **After Call Section**: Post-meeting automation and CRM updates
- **Trust Section**: Security and privacy features
- **Services Section**: Professional services offering
- **Pricing Section**: Three-tier pricing (Individual, Pro, Enterprise)
- **CTA & Footer**: Final call-to-action and navigation

## Design System

- **Primary Color**: Navy blue (`--cue-navy`)
- **Accent Color**: Green (`--cue-green`) 
- **Background**: Light gray sections for visual separation
- **Typography**: Serif headings, clean sans-serif body text
- **Fully responsive**: Mobile-first design

## Tech Stack

- Next.js 14
- Tailwind CSS
- Lucide React icons
